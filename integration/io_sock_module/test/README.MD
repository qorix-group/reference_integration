# Build and Test

The build and test instructions are provided in the file **build_and_test_instructions.txt** for the files ../src/mods-kso.c, tcp_test.rs and client.py.

The same instructions can be used to build & test the files stored in the sub-folders. The file ../src/mods-kso.c shall be replaced with *.c file from required sub-folder to build needed io-sock module.
